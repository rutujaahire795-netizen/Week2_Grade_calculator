# Week 2 - Student Grade Calculator

A simple Python program that calculates student grades based on marks.

## Features
- ✅ Grade calculation (A, B, C, D, F)
- ✅ Input validation (0-100 marks)
- ✅ Multiple student processing
- ✅ Class statistics (average, highest, lowest)
- ✅ Save results to file

## How to Run
```bash
python grade_calculator.py
